O que é uma API?
- API é a sigla em inglês para Application Programming Interface, que significa Interface de Programação de Aplicação. Trata-se de um conjunto de regras e protocolos que permitem que aplicativos de software se comuniquem entre si, trocando dados, recursos e funcionalidades.
